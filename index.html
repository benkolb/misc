The EUI colour scheme is: [191 208 234] (light blue), [56 139 199] (medium blue) and [0 84 130] (dark blue). Use this e.g. in Matlab by setting
EUIblues = [191 208 234; 56 139 199; 0 84 130]/255;
colormap(EUIblues);
